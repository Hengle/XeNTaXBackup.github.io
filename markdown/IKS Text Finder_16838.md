> IKS Text Finder is advanced tool for searching texts in files, filenames and directory texts. It also supports searching sets of bytes in binary files. It can be run on any folder at your computer, by easily clicking on the context menu option. Bytes can be inserted one by one or as a special text coded in ASCII-US, UTF8, UTF16 and other popular encodings. Tool supports dynamic progress bar and returning results any time you want. You can also see properties of the found file and open directory where the file was found. For binary files offsets of the found text are returned as decimal and hexadecimal value.
