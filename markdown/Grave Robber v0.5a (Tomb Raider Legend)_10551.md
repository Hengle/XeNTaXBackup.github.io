Grave Robber v0.5a

Current features :
- Extracts files from the bigfile archives
- Replaces files in the bigfile archive (same size or smaller only at the moment)
- Extracts files from DRM archives
- Converts RAW <-> TGA (with preview for both)
- Converts TRLTEX to TGA (with preview)
- Converts TRLMODEL files to correctly texturemapped OBJ (with 3D preview)

Useful in modding Tomb Raider Legend
