This is a little tool to extract .nub audio archives from audio .xfbin files for these games.

It seems every .xfbin file has only one .nub file inside. I have no information about other cases, so it will extract only one file.

Then you can process .nub file as usual with NUBExt.
[nar_nub.rar](https://xentaxbackup.github.io/file/9201_nar_nub.rar)
