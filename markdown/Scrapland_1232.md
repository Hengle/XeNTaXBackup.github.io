OK, here is the details of the format...

```
| American McGees Scrapland *.packed |
+------------------------------------+

4 - Header (Pozi)
4 - null
4 - Number Of Files?

// for each file
  4 - Filename Length
  X - Filename
  4 - File Length
  4 - File Offset
  
X - File Data
```


And here is the script...

```
Goto 8 0 ;
Get FNum Long 0 ;
For n = 1 to FNum ;
Get FNLen Long 0 ;
GetDString FN FNLen 0 ;
SavePos FOO 0 ;
Get FO Long 0 ;
SavePos FSO 0 ;
Get FS Long 0 ;
Log FN FO FS FOO FSO ;
Next n ;

```
