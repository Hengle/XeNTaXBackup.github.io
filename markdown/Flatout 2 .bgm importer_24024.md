I have added the Flatout 2 .bgm file format loader module to the following programs:

- 3D Object Converter v8.007	(Windows)
- i3DConverter v4.004		(macOS)
- i3DConverter v2.004		(Linux)

How to get the 3D Object Converter v8.007:
Download the 3D Object Converter from [ http://3doc.i3dconverter.com](http://3doc.i3dconverter.com) and install it or download and use the portable version.
After it just use the Help/Check for updates... function to get the v8.007.

How to get the i3DConverter macOS v4.004:
Download the i3DConverter from [ http://www.i3dconverter.com](http://www.i3dconverter.com) and install it.
After it just use the Help/Check for updates... function to get the v4.004.

How to get the i3DConverter Linux v2.004:
Download the i3DConverter from [ http://www.i3dconverter.com](http://www.i3dconverter.com) and install it.
After it just use the Help/Check for updates... function to get the v2.004.
