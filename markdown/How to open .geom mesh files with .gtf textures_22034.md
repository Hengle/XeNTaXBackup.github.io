- Post Title: How to open .geom mesh files with .gtf textures

You can open the EA Sports Fight Night Champion *.geom file using the following programs:

- 3D Object Converter v7.027 (Windows)
- i3DConverter v3.704 (macOS)
- i3DConverter v1.704 (Linux)

How to get the 3D Object Converter:
Download the 3D Object Converter from [ http://3doc.i3dconverter.com](http://3doc.i3dconverter.com) and install it or download and use the portable version.
After it just use the Help/Check for updates... function to get the v7.027.

How to get the i3DConverter macOS:
Download the i3DConverter from [ http://www.i3dconverter.com](http://www.i3dconverter.com) and install it.
After it just use the Help/Check for updates... function to get the v3.704.

How to get the i3DConverter Linux:
Download the i3DConverter from [ http://www.i3dconverter.com](http://www.i3dconverter.com) and install it.
After it just use the Help/Check for updates... function to get the v1.704.
