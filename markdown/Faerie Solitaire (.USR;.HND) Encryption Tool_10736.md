Tool for decrypt / encrypt USR and HND files.

PS: Output files just simple INI. Use any text editor 

USR - User Save Game
HND - Levels Data Scripts

```
        FSHNDDecrypt <pMode> <pInFile> <pOutFile>
```


```
        FSHNDDecrypt -hnd AgedCliffs_1.hnd AgedCliffs_1.ini
        FSHNDDecrypt -usr h4x0r.usr h4x0r.ini
```


```
        FSHNDDecrypt -hnd AgedCliffs_1.ini AgedCliffs_1.hnd
        FSHNDDecrypt -usr h4x0r.ini h4x0r.usr
```
