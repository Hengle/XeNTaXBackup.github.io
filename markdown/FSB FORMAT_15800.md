Hello,

If this is a .fsb file containing only one song, use Vgmstream. If it instead contains a pack of audio files, use Fsbii first to extract the individual audio files then convert each of them using the Vgmstream program.

Programs:
Vgmstream: [https://www.hcs64.com/vgmstream.html](https://www.hcs64.com/vgmstream.html)
Fsbii: [http://hcs64.com/files/fsbii08.zip](http://hcs64.com/files/fsbii08.zip)

Edit: Note that this will convert the audio to wav format, you will then have to use a separate program to convert them to mp3.
