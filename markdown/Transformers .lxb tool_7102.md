Simple use:
01.exe unpack the lxb file
02.exe repack the lxb file, based on the modified txt file

Both exe use Drag & Drop methode.
Or you can use command line too:
- 01.exe filename.lxb - unpack
- 02.exe filename.lxb - repack
[Transformers2.7z](https://xentaxbackup.github.io/file/4565_Transformers2.7z)
