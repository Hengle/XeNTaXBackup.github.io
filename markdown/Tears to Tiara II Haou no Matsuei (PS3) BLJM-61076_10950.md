This games files are encrypted with sdat encryption so you need a modded ps3 in order to decrypt these.
you need the latest version of decredat
[http://asmodean.reverse.net/pages/decredat.html](http://asmodean.reverse.net/pages/decredat.html)

this is what your list should look like just edit it to your needs

```
tree /dev_hdd0/GAMES/BLJM61076/PS3_GAME/USRDIR /dev_hdd0/GAMES/BLJM61076/PS3_GAME/USRDIR 00000000000000000000000000000000
#character folder
tree /dev_hdd0/GAMES/BLJM61076/PS3_GAME/USRDIR/Model/Character /dev_hdd0/GAMES/BLJM61076/PS3_GAME/USRDIR/Model/Character 00000000000000000000000000000000
#Map folder
tree /dev_hdd0/GAMES/BLJM61076/PS3_GAME/USRDIR/Model/Map /dev_hdd0/GAMES/BLJM61076/PS3_GAME/USRDIR/Model/Map 00000000000000000000000000000000
#weapons folder
tree /dev_hdd0/GAMES/BLJM61076/PS3_GAME/USRDIR/Model/Weapon /dev_hdd0/GAMES/BLJM61076/PS3_GAME/USRDIR/Model/Weapon 00000000000000000000000000000000
```
