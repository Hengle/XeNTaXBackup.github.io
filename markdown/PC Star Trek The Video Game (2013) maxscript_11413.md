Hi guys! Here's maxscript to import Star Trek: The Video Game models with bones (3ds max 2009-2012)
Here's tool by GMMan [viewtopic.php?f=32&t=10782](http://forum.xentax.com/viewtopic.php?f=32&t=10782) to extract game content from .cache archives

1. Load StarTrek_Bones.ms
2. Point it to a H.misc.cache file like KeenserBody_skel.fbx
3. A new dialogue shows up, this time choose KeenserBody_skel.fbx in B.misc.cache
Bones will be loaded

4. Load StarTrek_Models.ms
5. Point it to H.misc.cache file KeenserBody_skel.fbx
6. A new dialogue shows up, this time choose KeenserBody_skel.fbx in B.misc.cache
Model will be loaded above bones

Done

P.S. Report bugs, game isn't popular so if there will be no interest I'll leave script as it is.
