Here is an importer for textured, rigged and animated models from this game.
It requires Blender 249b and Python 2.6.6
How use:
1. run Blender249.blend
2. in Blender Text Window press alt+p and select:
- skn file - models. Script scanning which *.mot files use model for animation and copy all *.mot files in new folder 'anim_files'. It easy to find this mot files.
- mot file - use files from 'anim_files' in folder, where is a blend file. Mot files have the same number as used armature object.

Small update: fix bone count
