in blender 2.49 with a model loaded
press 'a' to select the model
press tab to change to edit mode
change to UV/Image editor
     (view picture or Shift + F10)
choose menu point Image then open
    to load a dds file
arrange it so that it fits the uv map
in 3D view change to Texture Paint mode
