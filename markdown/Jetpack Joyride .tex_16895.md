Version 129KB:

```
SHIFT: 52, so 0x33 header not 0x34

RGB 24 + 8 alpha
Width: 128 Height: auto

Flip RGB to BGR

```



Version 65KB:

```
SHIFT: 52 

RGB 12 + 4 alpha
Width: 128 Height: auto

Flip RGB to BGR

```
