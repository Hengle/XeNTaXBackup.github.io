I wasn't able to figure out if Paris is burning or not, but here is a script for quickbms to extract those archives, as also requested.

```
get UNKNOWN short	# 1
get UNKNOWN short	# 2
get TOTALSIZE long
get NULL long
get UNKNOWN short	# 0x10 - length of each entry...
get UNKNOWN short	# 5
get FILES short
get UNKNOWN short	# 0x10 - length of each entry...
get FNDIRSIZE long
get BLAAAAAAA long
math FNDIR == FILES
math FNDIR *= 16
math FNDIR += 32
log MEMORY_FILE FNDIR FNDIRSIZE
math DATAOFFSET == FNDIR
math DATAOFFSET += FNDIRSIZE

for i = 0 < FILES
	get FNPOS long
	get OFFSET long
	math OFFSET += DATAOFFSET
	get SIZE long
	get NULL long
	goto FNPOS MEMORY_FILE
	get NAME string MEMORY_FILE
	
	log NAME OFFSET SIZE
next i

```


I only had a look at one of the archives (DATA.HAB), but I'm confident it should work with the other of the games "HAB" files.
Having said that; dont hesitate to let me know if a problem arises with any of the others, or if you simply wan't to tell me I'm a douche (I don't mind that, either   )
