Files with this extension looks like database and dont contain files inside.

Here simple script for unpack BMg graphics if needed.

```
# 
# Written by Ekey (h4x0r)
# http://forum.xentax.com
# 
# script for QuickBMS http://quickbms.aluigi.org

get FILES short

for i = 0 < FILES
    get NSIZE byte
    getdstring NAME 23
    get OFFSET long
    get SIZE long
    log NAME OFFSET SIZE
next i
```
