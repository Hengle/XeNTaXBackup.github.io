Mobjects are effects with vertex animations, no skinned meshes or static.

Here is importer for textured effects from this game.
I use blender vertex animation. It is a cool thing.
It requires Blender version 249 and Python 26.

Steps:
1.unpack importer
2.run Blender249.blend
3.in Blender Text Window press alt+p and select mobject file.
4.play animation

Examples for testing:
