- Post Title: FFD Converter Tool (Ubisoft engine)

FFDConverter is a tool used to generate custom bitmap fonts for AnvilNext, Dunia, Disrupt engine (Far Cry, Assassin's Creed, Watch Dogs games and more... )

Currently the tool supports creating *.ffd | *.Fire_Font_Descriptor from:
Far Cry: 3, BloodDragon, Primal, 4, 5, New Dawn (for Far Cry 2 use FC2MFTConverter)
Assassin Creed: 2, Brotherhood, Revelation, 3, Liberation HD, 3 Remastered, 4, Rogue
Watch dogs: 1, 2, Legion
I Am Alive
Anno 2205

[Download](https://github.com/eprilx/FFDConverter/releases)
[Source](https://github.com/eprilx/FFDConverter)

Please upload more sample files (*.ffd | *.Fire_Font_Descriptor) from other games for this tool to support it.
